# BookFinder

## Hakkında
BookFinder, Google Kitaplar API'sini kullanarak belirtilen kitap adı ve yazarı ile eşleşen kitapları arayan basit bir Python scriptidir. Kullanıcıya, aradığı kitabın başlığı, yazarı ve kısa açıklaması gibi temel bilgileri sağlar. Bu araç, hızlı bir şekilde kitap araştırması yapmak isteyen kullanıcılar için idealdir.

## Kullanımı
Scripti çalıştırmak için Python yüklü bir sistem gereklidir. Terminal veya komut satırı aracılığıyla aşağıdaki komut ile scripti çalıştırabilirsiniz:

```
python book_finder.py "Kitap Adı" "Yazar İsmi"
```

Not: "Kitap Adı" ve "Yazar İsmi" yerine aramak istediğiniz kitabın adını ve yazarının ismini giriniz.

## Gereklilikler
- Python 3.x
- `requests` kütüphanesi (`pip install requests` ile yükleyebilirsiniz)
